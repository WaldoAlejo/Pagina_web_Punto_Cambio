## 💻 **Instrucciones de Uso**

### **Desarrollo Local:**
```bash
# 1. Instalar dependencias (ya hecho)
npm install

# 2. Iniciar servidor de desarrollo
npm run dev

# 3. Abrir en navegador
http://localhost:8080/
```

### **Producción:**
```bash
# 1. Construir para producción
npm run build

# 2. Vista previa local
npm run preview

# 3. Desplegar (copiar carpeta 'dist')
# Subir a hosting: Netlify, Vercel, AWS, etc.
```

### **Testing:**
```bash
# Ejecutar tests
npm run test

# Lint código
npm run lint
```
