# Punto Cambio - Sitio Web Oficial

![Punto Cambio](https://img.shields.io/badge/Version-2.0-gold)
![React](https://img.shields.io/badge/React-18.3.1-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.6.2-blue)
![Vite](https://img.shields.io/badge/Vite-6.0.11-purple)

Sitio web oficial de **Punto Cambio**, la casa de cambios líder en Ecuador con más de 25 años de experiencia en servicios transaccionales y cambio de divisas.

## 🌟 Características

### Servicios Principales
- 💱 **Cambio de Divisas** - Las mejores tasas del mercado ecuatoriano
- 📤 **Envío de Dinero** - Western Union autorizado
- 🪙 **Compra y Venta de Oro** - Transacciones seguras
- 🏢 **Franquicias** - Red de agencias en Ecuador
- 📦 **Courier** - Envío de paquetes nacional e internacional
- 💰 **Recaudaciones** - Sistema integral de cobros

### Características del Sitio
- ✅ Diseño moderno y elegante
- ✅ 100% Responsive (móvil, tablet, desktop)
- ✅ Calculadora de divisas en tiempo real
- ✅ Mapa interactivo de ubicaciones
- ✅ Formulario de contacto funcional
- ✅ Optimizado para SEO
- ✅ Botón flotante de WhatsApp
- ✅ Animaciones suaves con Framer Motion
- ✅ Sección de newsletter
- ✅ Accesibilidad mejorada

## 🚀 Tecnologías

- **Frontend Framework:** React 18.3.1
- **Language:** TypeScript 5.6.2
- **Build Tool:** Vite 6.0.11
- **Styling:** Tailwind CSS 3.4.17
- **UI Components:** Shadcn/ui con Radix UI
- **Animations:** Framer Motion 12.29.2
- **Icons:** Lucide React
- **Forms:** React Hook Form + Zod
- **HTTP Client:** React Query (TanStack)

## 📦 Instalación

```bash
# Clonar el repositorio
git clone <YOUR_GIT_URL>

# Navegar al directorio
cd Pagina_web_Punto_Cambio

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de producción
npm run preview
```
